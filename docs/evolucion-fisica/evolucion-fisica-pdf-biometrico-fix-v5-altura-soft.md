# Evolución física PDF biométrico - fix v5 altura soft

## Motivo
La silueta izquierda del PDF todavía se percibía ligeramente más baja que la derecha.

## Ajuste
Se incrementó la escala de la variante soft aproximadamente un 5% adicional y se recalibró su encuadre para mantener centrado y apoyo.

## Validación esperada
- altura más cercana entre ANTES y AHORA;
- proporción estable;
- plataforma y figura mejor alineadas.
