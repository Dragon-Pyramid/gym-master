# Evolución Física — Human Silhouette Scale & Position v7

## Objetivo

Aplicar el ajuste visual solicitado sobre la silueta humana de fondo:

- agrandarla aproximadamente un 30%;
- moverla un poco hacia la derecha;
- bajarla un poco.

## Cambios realizados

- Se incrementa el tamaño de la silueta base frontal en ~30%.
- Se desplaza horizontalmente la silueta hacia la derecha.
- Se desplaza verticalmente la silueta levemente hacia abajo.
- Se amplía el área de recorte para evitar cortes visuales al agrandar la imagen.
- No se modifica la lógica de interacción ni los hotspots; esta iteración se concentra únicamente en la posición/escala del background humano.

## Resultado esperado

La figura humana debería ocupar más presencia dentro del panel, verse mejor centrada respecto del overlay y alinearse de forma más natural con la lectura visual del usuario.
