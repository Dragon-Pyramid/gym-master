# Evolución Física — Human Silhouette Align v5

## Objetivo

Alinear mejor la capa geométrica interactiva con la silueta humana de fondo, tomando como referencia especialmente el tren inferior (muslos y pantorrillas), para que ambos sistemas visuales coincidan mejor.

## Cambios

- Se agranda levemente la silueta humana de fondo.
- Se desplaza y escala suavemente la capa de hotspots/interacción.
- La corrección toma como referencia visual principal la alineación de pantorrillas y muslos.
- Se mantiene la interactividad completa: slider, superpuesto, heatmap, selección de grupos, sexo masculino/femenino y vista frente/espalda.
- Se conserva el enfoque de hotspots sutiles para que la silueta siga siendo la protagonista.

## Resultado esperado

La geometría debería calzar mejor sobre la anatomía visible del background, reduciendo el desfasaje entre zonas clickeables y figura humana.
