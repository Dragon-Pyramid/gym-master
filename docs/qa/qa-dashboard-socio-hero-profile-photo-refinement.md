# QA Dashboard Socio — Ajuste visual de foto de perfil en inicio

## Rama

`feature/qa-dashboard-socio-recorrido-general`

## Objetivo

Mejorar la presentación visual del socio en el dashboard inicial después de validar que la captura de cámara funciona correctamente en "Editar mi perfil".

## Cambios

- Se agrandó la foto/avatar del socio en el ingreso del dashboard.
- Se redujo unos puntos la tipografía del nombre para equilibrar la composición visual.
- Se ajustó el layout del bloque de bienvenida para que la foto tenga mayor protagonismo sin romper mobile.

## Validación sugerida

1. Entrar como socio.
2. Verificar que la foto del socio se muestre más grande en el dashboard inicial.
3. Confirmar que el nombre no desborde y mantenga buena legibilidad.
4. Probar en desktop y mobile responsive.
5. Confirmar que "Editar mi perfil" conserva el flujo de subir foto y sacar foto.
