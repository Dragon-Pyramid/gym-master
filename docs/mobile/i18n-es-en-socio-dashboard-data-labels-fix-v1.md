# i18n ES/EN socio dashboard data labels fix v1

## Objetivo
Reducir textos de datos visibles que permanecían en español dentro del dashboard mobile del socio cuando el idioma activo era Inglés.

## Ajustes
- `PLAN AUTOMATICO - DEFINICIÓN` se normaliza visualmente como `Automatic plan - Definition`.
- `FUNCIONAL TARDE` se normaliza visualmente como `Functional afternoon`.
- `YOGA INICIAL` se normaliza visualmente como `Beginner Yoga`.
- Se mantiene el dato original intacto; el cambio es solo de presentación en la UI.

## Nota técnica
Estos textos provienen de datos cargados o generados previamente, no del copy fijo de interfaz. Esta normalización cubre los casos visibles detectados en QA. Para una internacionalización completa de contenido editable por administración, se recomienda evaluar campos bilingües o traducción asistida en una feature futura.
