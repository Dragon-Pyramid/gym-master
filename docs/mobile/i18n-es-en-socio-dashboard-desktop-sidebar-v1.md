# i18n ES/EN socio dashboard desktop sidebar v1

## Objetivo
Completar la validación web/desktop del dashboard del socio dentro de `feature/i18n-es-en-socio-dashboard-v1`, cubriendo el menú lateral usado cuando el socio accede desde PC.

## Alcance
- Traducción ES/EN de grupos del sidebar del socio.
- Traducción ES/EN de items visibles del menú del socio.
- Traducción del botón de cierre de sesión.
- Traducción de labels accesibles del menú mobile/desktop.

## Notas
- No cambia permisos, keys ni rutas del menú.
- La traducción es solo de presentación visual.
- No toca DB, endpoints ni Swagger/OpenAPI.
