# i18n ES/EN socio dashboard data labels fix v2

## Objetivo
Completar la normalización visual de datos cargados/generados previamente que todavía aparecían en español dentro del dashboard mobile del socio cuando el idioma activo era Inglés.

## Ajustes realizados
- Dietas:
  - `Plan automático - Definición` se muestra como `Automatic plan - Definition`.
  - `Definición` se muestra como `Definition`.
- Actividades:
  - `Functional tarde` / `Funcional tarde` se muestra como `Functional afternoon`.
  - `Yoga inicial` se muestra como `Beginner Yoga`.
  - Se agregan normalizaciones para `mañana`, `tarde`, `noche`, `inicial`, `intermedio` y `avanzado`.

## Nota
El dato original no se modifica en base de datos. La normalización ocurre solo a nivel de presentación mobile del socio.
