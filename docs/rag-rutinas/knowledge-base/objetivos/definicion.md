# Objetivo: Definición

## Propósito

Reducir grasa corporal preservando masa muscular y rendimiento.

## Criterios de entrenamiento

- Mantener ejercicios de fuerza/hipertrofia.
- Evitar transformar toda la rutina en cardio.
- Incluir cardio o trabajo metabólico según tolerancia.
- Usar descansos moderados y volumen sostenible.
- Priorizar técnica y recuperación por déficit calórico.

## Orden de sesión

```txt
fuerza/compuesto → hipertrofia secundaria → accesorios → cardio/metabólico opcional
```

## Dieta base

- Déficit calórico moderado.
- Alta proteína.
- Carbohidratos estratégicos alrededor del entrenamiento.
- Control de saciedad y adherencia.

## Errores a evitar

- Volumen excesivo con déficit alto.
- Sacar todos los ejercicios pesados.
- Cardio antes de ejercicios técnicos pesados si afecta rendimiento.
- Rutinas idénticas a volumen cambiando solo repeticiones sin lógica.
