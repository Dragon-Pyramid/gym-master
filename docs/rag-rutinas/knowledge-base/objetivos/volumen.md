# Objetivo: Volumen

## Propósito

Ganar masa muscular con superávit calórico controlado y progresión de cargas/volumen.

## Criterios de entrenamiento

- Priorizar hipertrofia.
- Combinar ejercicios compuestos y accesorios.
- Trabajar grupos musculares 1-2 veces por semana según días disponibles.
- Usar rangos de 6-12 repeticiones en principales y 10-20 en accesorios.
- Controlar fatiga para sostener progresión mensual.

## Orden de sesión

```txt
compuesto principal → compuesto secundario → accesorio multiarticular → aislamiento → core/opcional
```

## Intensidad

| Nivel | RPE |
|---|---|
| Inicial | 6-7 |
| Intermedio | 7-8 |
| Avanzado | 8-9 |

## Dieta base

- Superávit moderado.
- Proteína suficiente.
- Carbohidratos adecuados para rendimiento.
- Control de ganancia de grasa.

## Errores a evitar

- Exceso de volumen sin recuperación.
- Rutinas de 6 días con poca cobertura de piernas/espalda.
- Repetir la misma rutina mensual sin progresión.
- Usar solo ejercicios aislados para grupos grandes.
