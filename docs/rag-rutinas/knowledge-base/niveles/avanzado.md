# Nivel Avanzado

## Perfil

Socio con experiencia, técnica y mayor tolerancia al volumen/intensidad.

## Criterios

- Mayor especialización.
- RPE 8-9 en ejercicios seleccionados.
- Mayor volumen por grupo.
- Variantes específicas y control de fatiga.
- Puede usar frecuencia 2 si la recuperación lo permite.

## Programación

- Ejercicios compuestos priorizados.
- Variantes por ángulo/patrón.
- Accesorios para puntos débiles.
- Deload o descarga cada 6-8 semanas si hay fatiga.

## Salida esperada del RAG

La rutina debe estar más periodizada, con lógica de progresión y control de fatiga. No debe ser solo “más ejercicios”.
