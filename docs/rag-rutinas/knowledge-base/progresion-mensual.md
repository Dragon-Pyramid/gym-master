# Progresión mensual

## Propósito

Definir cómo una rutina nueva debe evolucionar respecto de la anterior.

## Regla base

Un socio normalmente cambia o ajusta rutina por mes. La nueva rutina debe conservar continuidad, pero no ser idéntica.

## Tipos de progresión

- Aumentar carga si se completan repeticiones objetivo.
- Aumentar repeticiones dentro del rango.
- Aumentar una serie en ejercicios prioritarios.
- Cambiar variante de ejercicio.
- Cambiar orden dentro de una lógica segura.
- Reducir descanso para resistencia/definición si aplica.
- Mejorar técnica/tempo antes de subir carga.

## Ejemplo

Mes 1:

```txt
Press banca plano 4x8
Press inclinado mancuernas 3x10
Aperturas polea 3x12
```

Mes 2 posible:

```txt
Press inclinado barra 4x6-8
Press plano mancuernas 3x8-10
Cruces polea 3x12-15
```

No es idéntico, pero mantiene objetivo y grupo muscular.

## Reglas de seguridad

- No progresar carga si la técnica empeora.
- No subir volumen si hay dolor o fatiga persistente.
- No repetir exactamente estructura completa del mes anterior.
- En definición/bajar de peso, controlar fatiga por déficit calórico.
